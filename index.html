Red Vs Wolf Instructions

How To Start:

To start the game click the "Start Game" button in the game window.

Controls
Use the keyboard and mouse for game input.
  • R Key - Press the R key to roll the dice;
  • Mouse Click - Use the mouse key to make a selection/choice;
Game Play

Arrive at grandmother's house before the wolf beats you to it.

Additions to game engine
  • Alternate Endings - Depending on the decisions made from the events triggered a different ending may be displayed for the winner.
  • Frame by Frame Animation - The images for an object is animated as if it's shuffling. In the game the animation is displayed for 3 seconds. Example rolling dice, and revealing of the traps and event.
  • Time based animation (Updated) - This is applied to both characters where it moves across the board automatically now vertically as well. Previously the image/character/enemy only moved horizontally diagonally as seen in the aliens for Candy4Aliens.

Traps

When landing on a tile with a trap it triggers one of 3 events for that character.

Move Back 1
Character is force to go back 1 square. 		Move Back 2
Character is force to go back 2 square. 		Move Back 3
Character is force to go back 3 square.
Events

When landing on a tile with a scroll it triggers one of 4 events for that character. There are 4 different events per character. Decisions made also effect the ending story outcome.

Red's Event
Red gets to make the decision of what it wants to do. 		Wolf's Event
The wolf gets to make the decision of what it wants to do.
